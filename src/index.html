Kitchen Table Mode Tracker

The Kitchen Table Mode Tracker (KTM Tracker) is a platform dedicated to creating and managing tournaments for the game Splinterlands. This platform allows players to:

  • Organize tournaments with predefined customizable rule sets, card sets, and level caps as the game provides.
  • Join active tournaments and participate in structured competitions.
  • Automatically match players into brackets and dynamically update results.

Since the KTM Tracker is a third-party platform, all battles must be submitted manually through Splinterlands' Challenge interface. It is the player's responsibility to ensure that challenges are submitted and accepted in accordance with the tournament's predefined rules.
